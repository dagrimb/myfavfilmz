API Endpoints, Requests and Responses

What the endpoint returns Endpoint URL Method Type Format of Request Data Format of Response Data
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies.
Return movie description /movies/:name/description GET None A JSON object holding data about a movie’s description, containing the movie name, year it was released and a description.

{
Movie : “Rochelle Rochelle”,
Details : {
Year Released : 1990,
Description: “A young woman’s strange, erotic journey from Milan to Minsk.”
}
}
Return genre /movies/:name/genre GET None A JSON object holding data about a movie’s genre and a description of the genre.

{
Movie: “Meet the Parents”,
Details : {
Genre: “Comedy,”
Genre Info: “[Info about “comedy” genre]
}
}
Return director /movies/:name/director GET None A JSON object holding data about a movie’s director (bio, birth year, death year).

{
Director : “Martin Scorsese”,
Details : {
Bio : “Martin Scorsese was born…”
Birth Year : 1942 (78 y/o)
}
}
Return imageURL /movies/:name/ImagePath GET None A JSON object holding data about a movie’s poster and images.

{
Movie: “The Shining”
Images: {
imageURL: https://encrypted-tbn1.gstatic.com/images?q=tbn:ANd9GcTfWqUZeOS0kkid0unmHjF3KCG64pQJYvqKxyN0T3hgBuYUyB-_ ….
}
}
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{
name :
“Dustin Vargas”
} 		A JSON object holding data about the account that was added.

{
id: 7,
name: “Dustin Vargas”,
favorite movies: {
The Sound of Music,
Chariots of Fire
}
}
Allow new users to update /users/:userId PUT None A text message indicating that the username was updated.
Allow users to add a movie to their list of favorites /users/:userId/FavoriteMovies POST None A JSON object holding data about the movie that was added, including an ID.

{
Id: 5
Movie : “The Green Mile”
}
Allow users to remove a movie from their list of favorites /users/:userId/FavoriteMovies/:movieId DELETE None A text message indicating whether the movie was successfully removed.
Allow existing users to deregister /users/:name DELETE None A text message indicating whether the user was successfully removed.
