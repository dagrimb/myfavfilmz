Return description of a single movie to the user /movies/:Title/Description GET None A JSON object holding data about a movie’s description, containing the movie name, year it was released and a description.

Example (for Lethal Weapon)
{
"Synopsis": "Two newly paired cops who are complete opposites must put aside their differences in order to catch a gang of drug smugglers.",
"Source": "https://www.imdb.com/title/tt0093409/?ref_=nv_sr_srsg_3"
}
Return genre information of a single movie to the user /movies/:Title/Genre GET None A JSON object holding data about the movie’s genre (the name of the genre, a description of the genre and the IMDB site that description was retrieved from)

Example (for Face/Off)
{
"Name": "Action",
"Description": "Action film is a film genre in which the protagonist or protagonists are thrust into a series of events that typically include violence, extended fighting, physical feats, rescues and frantic chases.",
"Source": "https://en.wikipedia.org/wiki/Action_film"
}
Return director information of a single movie to the user /movies/:Title/Director GET None A JSON object holding data about the movie’s director (the name of the director, a brief bio, the year of their birth, the year of their death (if applicable) and the Wikipedia site that description was retrieved from).

Example (for Get Shorty)

"Bio": "Barry Sonnenfeld is an American filmmaker and television director.",
"Birth": "1953",

Return imageURL of a single movie to the user /movies/:Title/Image GET None A link to an image (in most cases, the movie poster) for a movie.

Return a list of ALL actors to the user /actors GET None A JSON object holding data about all actors.

Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{
"Username": " ",
"Password": " ",
"Email": " ",
"Birthday": " ",
"FavoriteMovies": [ ]
}
 A JSON object holding data about the account that was added.

{
"FavoriteMovies": [ ],
"_id": "[myfavfilmz generated unique id]",
"Username"(example): "mrbeanfan222",
"Password"(example): "222nafnaebrm",
"Email"(example): "mrbeanfan222@gmail.com",
"Birthday": "YYYY-MM-DD",
}
Return data about a single user by userId /users/:userId GET None A JSON object holding data about a user (see previous POST request).
Allow new users to update their username /users/:userId PUT None A JSON object holding data about the account that was updated.

{
"FavoriteMovies": [ ],
"_id": [myfavfilmz generated unique id],
"Username": "mrlegumefan20",
"Password": null,
"Email": null,
"Birthday": null,
}
Allow users to add a movie to their list of favorites /users/:userId/Movies/:MovieID POST None A JSON object holding data about user, showing the new movie added to their list of favorites

{
"FavoriteMovies": [
"604ab711e2b77c5f801cf1ba"
],
"_id": [myfavfilmz genereated unique id],
"Username": "mrlegumefan20",
"Password": null,
"Email": null,
"Birthday": null,
}
Allow users to remove a movie from their list of favorites /users/:userId/Movies/:MovieID DELETE None A JSON object holding data that gives and updated listed of favorite movies after movie has been removed
Allow existing users to deregister /users/:userId DELETE None A text message indicating whether the user was successfully removed.
